How to Use This Dashboard

  1. Upload Data: Click "Choose File" and select your CSV file containing network traffic data.
  2. Analyze: Click "Upload & Analyze" to run the machine learning models.
  3. View Results: Navigate to the Results page to see model performance metrics.
  4. Compare Models: Use the interactive charts to compare different models.
  5. Export: Download reports and visualizations for your reference.

Data Format Requirements

Your CSV file should include network traffic features and a 'class' column identifying normal/attack traffic.
